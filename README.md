# Перевірка джерел дисертації

Веб-застосунок для автоматичного виявлення невикористаних бібліографічних джерел у тексті дисертації.

## Можливості

- Завантаження `.pdf` і `.docx` файлів (до 30 МБ)
- Автоматичне знаходження розділу «Список літератури»
- Ручний режим вказівки розташування списку
- Підтримка багаторядкових записів і двох форматів нумерації (`1.` та `[1]`)
- Розгортання діапазонів посилань (`[15–18]` → 15, 16, 17, 18)
- Таблиця сиріт з повним текстом кожного джерела
- Рядок номерів для копіювання в одне поле

## Структура проєкту

dissertation-checker/

## Локальний запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой на Streamlit Community Cloud

1. Завантажити репозиторій на GitHub (публічний або приватний).
2. Відкрити [share.streamlit.io](https://share.streamlit.io) → **New app**.
3. Обрати репозиторій, гілку `main`, файл `app.py`.
4. Натиснути **Deploy**.

Ресурси безкоштовного тіру (~1 ГБ RAM) достатньо для обробки дисертацій до 30 МБ.

## Ліміти та обмеження

| Параметр | Значення |
|---|---|
| Максимальний розмір файлу | 30 МБ |
| Скан-PDF (без текстового шару) | Не підтримується |
| Пакетна обробка | Не підтримується |
| Експорт результатів | Не підтримується |

## Формат посилань (підтримується)

| Приклад | Результат |
|---|---|
| `[1]` | джерело 1 |
| `[1, 15]` | джерело 1 (15 — сторінка) |
| `[1; 3; 7]` | джерела 1, 3, 7 |
| `[15-18]` або `[15–18]` | джерела 15, 16, 17, 18 |
| `[1, 15; 3; 5, 20-25]` | джерела 1, 3, 5 |
